### Admin Dashboard

A Dashboard made using React


## Features

* Supports different browsers: IE10,Chrome,FireFox using `react-app-polyfill`.
* Interactive Charts.
* Internationalization using `react-i18next`. 
* Integrated `Redux` for setting up global Internationalization.
